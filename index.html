DP-Util

Radware DefensePro Remote Configuration and Scripting Utility
This utility allows the remote configuration of commonly used modules on the Radware DefensePro.
To get the latest information, open tickets and view current status - please visit the project page on GitHub.

Download

The latest version included in the following packages is DP-util 0.3.

Binary packages

Binary packages for Windows, Linux and OS X can be found here.
Please be sure to read the README.txt file prior to using this utility.

Source package
